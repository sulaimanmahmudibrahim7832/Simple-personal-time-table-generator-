Simple Personal Time Table Generator
A web-based application that allows students to create and manage their personal timetable. Users can input their details, add courses with specific days and times, and generate a structured, printable timetable.
The project demonstrates core frontend concepts such as DOM manipulation, event handling, dynamic table rendering, and local storage for data persistence. It also includes features like course management (add/delete), profile image preview, and a print-ready layout.
Built using HTML, CSS, and JavaScript as a beginner-level project to practice building interactive web applications.
